Unity Model DESCRIPTION: This section will describe the model and what will be happening to it. Controls for the user will be described in the next section.


Model Controls

  • Keyboard Toggle Control 'w': Pressing key 'w' will rotate the model vertically upwards.
  • Keyboard Toggle Control 's': Pressing key 's' will rotate the model vertically downwards.
  • Keyboard Toggle Control 'a': Pressing key 'a' will rotate the model horizontally to the left.
  • Keyboard Toggle Control 'd': Pressing key 'd' will rotate the model horizontally to the right.
  • Button 'solid': Pressing the solid button will annotate a filled in hollow cathode.
  • Button 'expand': Pressing the double arrays in the bottom right will expand to full screen mode.
  • Button 'transparent': Pressing the transparent button will annotate a transparent look of the hollow cathode.
  • Mouse Upward Scroll: Scroll upward on mouse to zoom in.
  • Mouse Downward Scroll: Scroll downward on mouse to zoom out.
  • Keyboard Toggle Control 'esc' : Exits out of full screen mode.
